# devbox

## Features

- NvChad support
- tmux
- oh-my-zsh
- vs-code-js-debug

## How to use

From terminal:

`apt update && apt install -y curl && bash <(curl -s https://raw.githubusercontent.com/gpt-partners/dotfiles/main/install.sh)`

Or as Docker container:

```
docker-compose up -d --build
ssh -p 23 root@devbox
```
